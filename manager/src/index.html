api接口

📊 获取服务状态

GET /api/status

获取 Daily 服务的当前运行状态

show details

使用 curl：

curl http://127.0.0.1:11452/api/status

使用 PowerShell： 

Invoke-RestMethod -Uri "http://127.0.0.1:11452/api/status"

响应示例： 

{
  "enabled": true,
  "process_status": "running"
}

🔄 启用/禁用服务

POST /api/toggle?enabled=true

控制 Daily 服务的启用或禁用状态

show details

启用服务 - curl：

curl -X POST "http://127.0.0.1:11452/api/toggle?enabled=true"

禁用服务 - curl： 

curl -X POST "http://127.0.0.1:11452/api/toggle?enabled=false"

使用 PowerShell： 

Invoke-RestMethod -Uri "http://127.0.0.1:11452/api/toggle?enabled=true" -Method POST

临时设置壁纸

POST /api/set_wallpaper?url=<图片URL>

从指定 URL 下载图片并立即设置为桌面壁纸

show detail

使用 curl：

curl -X POST "http://127.0.0.1:11452/api/set_wallpaper?url=https://example.com/wallpaper.jpg"

使用 PowerShell： 

$url = "https://example.com/wallpaper.jpg"
Invoke-RestMethod -Uri "http://127.0.0.1:11452/api/set_wallpaper?url=$url" -Method POST

获取日志

GET /api/logs/manager 或 /api/logs/daily

获取 Manager 或 Daily 程序的日志内容

show detail

获取 Manager 日志 - curl： 

curl http://127.0.0.1:11452/api/logs/manager

获取 Daily 日志 - curl： 

curl http://127.0.0.1:11452/api/logs/daily

使用 PowerShell 并保存到文件： 

Invoke-RestMethod -Uri "http://127.0.0.1:11452/api/logs/daily" | Out-File daily.log

获取配置文件

GET /api/config

获取 Daily 当前使用的配置文件内容（JSON 格式）

show details

使用 curl：

curl http://127.0.0.1:11452/api/config

使用 PowerShell 并格式化输出： 

Invoke-RestMethod -Uri "http://127.0.0.1:11452/api/config" | ConvertTo-Json -Depth 10

设置本地配置

POST /api/set_local_config

设置自定义本地配置文件及截止日期

show details

使用 curl：

curl -X POST "http://127.0.0.1:11452/api/set_local_config" \
  -H "Content-Type: application/json" \
  -d '{
    "json_content": "{\"wallpapers\":{\"days\":{},\"dates\":{},\"periods\":[]}}",
    "expire_date": "2024-12-31"
  }'

使用 PowerShell： 

$body = @{
    json_content = '{"wallpapers":{"days":{},"dates":{},"periods":[]}}'
    expire_date = "2024-12-31"
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://127.0.0.1:11452/api/set_local_config" `
    -Method POST -Body $body -ContentType "application/json"
